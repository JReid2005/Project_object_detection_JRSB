# Project_object_detection_JRSB
Object detection workflow using NVIDIA Jetson Nano and GPU acceleration

## Problem Statement
A security camera monitors multiple moving objects. For each frame, features such as size, speed, and brightness are measured and ranked to determine which objects should be tracked.

## Hardware
- NVIDIA Jetson Nano
- Camera

## Software
- Python
- CUDA
- OpenCV
- PyTorch

## Object Detection Model
YOLOv5

## Techniques
- Brent-Kung scan algorithm

## Optimization Techniques
- Tiling
- Thread coarsening
